# Danh sách lệnh npm install cho FE

Dưới đây là các lệnh npm install cần thiết để cài đặt toàn bộ các package cho phần FE của dự án này.

## Cài đặt tất cả dependencies và devDependencies

```bash
npm install
```

## Hoặc cài đặt từng package riêng lẻ

### Dependencies

```bash
npm install @vercel/edge-config axios chart.js cors dayjs html-to-image html2canvas jspdf react react-chartjs-2 react-dom react-icons react-router-dom
```

### DevDependencies

```bash
npm install -D @eslint/js @tailwindcss/cli @tailwindcss/postcss @types/react @types/react-dom @vitejs/plugin-react autoprefixer eslint eslint-plugin-react-hooks eslint-plugin-react-refresh globals postcss tailwindcss vite
```

> **Lưu ý:**
>
> - Chạy `npm install` trong thư mục FE để tự động cài đặt tất cả các package trên.
> - Nếu bạn muốn cài đặt từng package riêng lẻ, hãy sử dụng các lệnh ở trên.
